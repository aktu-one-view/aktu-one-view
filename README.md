## üõ†Ô∏è Quick Start Guide
### Accessing the Portal
1. Head to **[oneview.aktu.ac.in](https://oneview.aktu.ac.in)** or **[erp.aktu.ac.in](https://erp.aktu.ac.in)** (verify via [aktu.ac.in](https://aktu.ac.in)).
2. For One View: Enter your roll number ‚Üí Select "One View Display of Student Data" ‚Üí Pick semester.
3. For ERP: Use User ID & Password ‚Üí Navigate to results/fee sections.
4. Download/print marksheets directly.

### Checking Results
1. Log in as above.
2. Choose semester (1-8).
3. View details & hit download for PDF.
4. Spot issues? Apply re-evaluation or carry-over via ERP.

### Clearing Backs
- Log into ERP ‚Üí Submit carry-over form (‚Çπ1500/paper).
- Prep with PYQs from [Collgo](https://collgo.com/aktu-pyq).
